## Install

```
npm install --save-dev react-gridview
```

## Usage

### ・ES6

```
// .jsx
import {GridView} from "react-gridview";

<GridView />
```

### ・TypeScript
```
// tsconfig.json
{
    "compilerOptions": {
        "moduleResolution": "node",
        // --- ohter options
    }
}
```
  
```
// .tsx
import {GridView} from "react-gridview";

<GridView />
```
